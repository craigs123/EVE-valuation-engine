2. Pixel Extraction Function
async function extractPixelValue(assetUrl: string, lat: number, lon: number): Promise<number | null>
Key Steps:

Open COG from URL: const tiff = await fromUrl(assetUrl);
Get image bounds: const [minX, minY, maxX, maxY] = image.getBoundingBox();
Coordinate transformation: Convert lat/lon to pixel coordinates
Pixel read: Extract value using window-based raster reading
Python Implementation Guide:

import rasterio
from rasterio.windows import Window
import requests
def extract_pixel_value(asset_url: str, lat: float, lon: float) -> float:
    # Open COG directly from HTTP URL
    with rasterio.open(asset_url) as dataset:
        # Get geographic bounds
        bounds = dataset.bounds
        width, height = dataset.width, dataset.height
        
        # Check coordinate bounds
        if not (bounds.left <= lon <= bounds.right and 
                bounds.bottom <= lat <= bounds.top):
            return None
            
        # Transform geographic coordinates to pixel coordinates
        pixel_x = int((lon - bounds.left) / (bounds.right - bounds.left) * width)
        pixel_y = int((bounds.top - lat) / (bounds.top - bounds.bottom) * height)
        
        # Read pixel value using window
        window = Window(pixel_x, pixel_y, 1, 1)
        pixel_value = dataset.read(1, window=window)[0, 0]
        
        return pixel_value
3. Coordinate Transformation Details
Mathematical conversion from geographic to pixel coordinates:

const pixelX = Math.floor(((lon - minX) / (maxX - minX)) * width);
const pixelY = Math.floor(((maxY - lat) / (maxY - minY)) * height);
Important Notes:

Y-axis is flipped in raster coordinates (maxY - lat)
Use Math.floor() to get integer pixel indices
Bounds checking prevents out-of-range access
